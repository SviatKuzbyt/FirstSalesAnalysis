# First Sales Analysis
Analysis of open data set sales on retail sales performance

### Progress of work
1. Data cleaning
2. Creating data for analysis: sales by period, category, region and beyond
3. Data analysis and the production of conclusions and recommendations

### Work result
As a result of the analysis, grouped tables and graphs were created, placed in `Result/Report.docx` and conclusions and recommendations, placed in `Result/Summary.docx`

### Resources used
- Data set: [retail-sales-dataset](https://www.kaggle.com/datasets/abbas829/retail-sales-dataset)
- Python
- Pandas
- Matplotlib
- MS Excel, Word
